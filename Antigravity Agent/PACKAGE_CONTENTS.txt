KNOWN LIMITATIONS & NOTES
========================================

Google Apps Script Limits:
  • Scripts must complete within 6 minutes (timeout)
  • Storage limited to 15 GB per account
  • API rate limits apply (varies by service)
  • No direct server-side session storage
  • No file write permissions outside Drive

Design Decisions:
  • Using Google Sheets as database (no SQL)
  • No mobile app (web app only)
  • No real-time collaboration (refresh needed)
  • No offline mode
  • Single deployment per project
